# Basic Calculator

This project is a simple web-based calculator that allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. It features a user-friendly interface with number keys, operation buttons, and a clear button.

## Project Structure

```
calculator
├── index.html        # Main HTML document for the calculator
├── styles            # Directory containing CSS styles
│   └── style.css     # Styles for the calculator interface
├── scripts           # Directory containing JavaScript files
│   └── calculator.js  # JavaScript code for calculator functionality
└── README.md         # Documentation for the project
```

## How to Use

1. Open the `index.html` file in a web browser.
2. Use the number keys to input numbers.
3. Click on the arithmetic operation buttons to perform calculations.
4. Use the clear button to reset the calculator.

## Dependencies

This project does not have any external dependencies. It uses standard HTML, CSS, and JavaScript.

## Running the Project

Simply open the `index.html` file in your preferred web browser to start using the calculator. No additional setup is required.
